# Desktop Cleaner — Инструкция по установке (v2, pywebview UI)

## Что нужно установить

1. **Python 3.10+** — https://www.python.org/downloads/
   - При установке поставь галочку ✅ "Add Python to PATH"
2. На Windows для pywebview нужен **Microsoft Edge WebView2 Runtime** — на Windows 10/11
   он почти всегда уже встроен в систему. Если приложение не запускается с ошибкой про
   WebView2, скачай runtime: https://developer.microsoft.com/microsoft-edge/webview2/

---

## Установка зависимостей

```
pip install -r requirements.txt
pip install pyinstaller
```

---

## Запуск без сборки

```
python desktop_cleaner.py
```

Иконка появится в трее. Двойной клик — открывает новое окно настроек на pywebview
(тёмная/светлая тема, стеклянная шапка, переключатель тем — ☀️/🌙 в правом верхнем углу).

---

## Сборка в .exe

```
pyinstaller DesktopCleaner.spec
```

Готовый `.exe` будет в `dist/DesktopCleaner.exe`. Spec уже настроен так, чтобы
подтянуть папки `ui/` и `assets/` внутрь сборки — без них окно настроек не откроется.

---

## Структура проекта (важно: всё в одной папке, без подпапок!)

```
desktop_cleaner.py     — вся логика (конфиг, перенос файлов, трей, автозапуск, веб-сервер)
stats.py                — журнал и счётчик обработанных файлов
index.html              — разметка окна настроек
style.css               — тёмная/светлая тема, стекло, металл, оранжевый акцент
                           (картинки листа и металла встроены прямо в CSS как base64,
                           отдельных .jpg файлов больше нет — нечему потеряться при копировании)
app.js                   — связь UI <-> Python через pywebview JS API
```

**Все 7 файлов должны лежать в одной папке рядом** — никаких подпапок `ui/` или `assets/`
больше не нужно, именно из-за этого была ошибка 404 (при скачивании файлов по одному
вложенная структура папок терялась).

## Что хранится на диске

```
C:\Users\<Пользователь>\AppData\Roaming\DesktopCleaner\config.json   — настройки
C:\Users\<Пользователь>\AppData\Roaming\DesktopCleaner\stats.json    — журнал обработки
```

---

## Известные ограничения этой версии

- Тестировал синтаксис кода (Python и JS компилируются без ошибок), но **визуально
  на реальном Windows-десктопе с WebView2 не проверял** — это нужно сделать на твоей машине.
- Тема (светлая/тёмная) запоминается в localStorage внутри окна, а не в config.json —
  при желании можно перенести в общий конфиг, чтобы синхронизировать с другими настройками.
- `pick_folder()` использует диалог pywebview — на некоторых сборках Windows может
  потребоваться более новая версия WebView2 Runtime.
